Difference between Session Storage and Local Storage

Session Storage

  • Temporary storage
  • Data lost on browser close
  • Limited to 5MB storage
  • Accessible only in current tab/window
  • Stored in memory
  • Suitable for temporary data
  • Data cleared on session end
  • Faster access times
  • Suitable for small amounts of data
  • Not secure for sensitive data
  • Session Storage Example

Local Storage

  • Persistent storage
  • Data persists on browser close
  • Larger storage capacity (10MB)
  • Accessible across multiple tabs/windows
  • Stored in file or database
  • Suitable for long-term data
  • Data remains until explicitly cleared
  • Slower access times
  • Suitable for larger amounts of data
  • More secure than session storage
